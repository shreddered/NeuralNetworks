# Лабораторные работы по курсу ИТИБ

Лабораторные работы на _Haskell_ представляют собой _cabal_-пакеты.
Собрать их можно следующим образом:

```bash
$ cabal build
```

Запуск, в свою очередь, делается следующим образом:

```bash
$ cabal run
```
